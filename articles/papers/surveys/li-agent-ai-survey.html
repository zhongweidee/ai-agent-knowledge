Agent AI: Surveying the Horizons of Multimodal Interaction

综述 2024 李飞飞

作者: Fei-Fei Li 等 (Stanford & Microsoft)
链接: arXiv:2409.09030

核心贡献

80 页纲领性综述，提出 Agent AI 的 五模块认知架构：

  1. 环境与感知 — 多模态输入处理（视觉、语言、音频等）
  2. 认知 — LLM/VLM 作为推理核心
  3. 行动 — 工具调用、环境交互
  4. 学习 — 从交互中持续改进
  5. 记忆 — 短期/长期记忆管理

关键洞察

  • 将分散的技术线索统一到多模态 Agent 视角
  • 建立了感知-决策-行动闭环框架
  • 强调 Agent 需要具备物理世界和数字世界的双重交互能力

主要内容

  1. 引言: 从 LLM 到 Agent AI 的演进
  2. Agent AI 基础: 核心概念与设计原则
  3. 多模态交互: 视觉、语言、音频等多通道融合
  4. 学习与适应: 在线学习、强化学习、少样本学习
  5. 应用案例: 机器人、游戏、医疗、教育等
  6. 挑战与未来方向: 安全、可解释性、泛化能力

这篇综述的独特价值在于它不局限于纯语言 Agent，而是将视觉、机器人等多模态领域统一到 Agent AI 框架下。适合作为知识体系的第一站。
