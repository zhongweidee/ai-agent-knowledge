# 学习路线

> 按顺序从入门到精通。

## 第一阶段：建立基础

1. 读 [Agent AI 综述](01-papers/surveys/li-agent-ai-survey.md)（李飞飞） → 建立全景图
2. 读 [ReAct 论文](01-papers/classics/react-synergizing-reasoning-acting.md) → 理解 Agent 核心循环
3. 了解 [术语表](04-glossary/glossary.md) → 扫清概念障碍

## 第二阶段：深入核心

1. 读 **CoALA** → 认知架构理论
2. 读 [Generative Agents](01-papers/classics/generative-agents-stanford-town.md) → 多智能体+记忆
3. 读 **Model-Native 综述** → 前沿方向
4. 读 **CoT / ToT / Reflexion** → 推理技术

## 第三阶段：动手实践

1. 读 [Building Effective Agents](03-tutorials/building-effective-agents-anthropic.md)
2. 选一个[框架](02-frameworks/framework-comparison.md)动手
3. 从 Prompt Chaining 开始，逐步增加复杂度

## 第四阶段：进阶

1. 多智能体系统（AutoGen / CrewAI）
2. Agent 安全与评测
3. MCP / A2A 协议
4. 生产化部署
