# 术语表

> AI Agent 领域常见术语速查。

| 术语 | 英文 | 说明 |
|------|------|------|
| **Agent** | Agent | 能感知环境、自主决策、采取行动的 AI 系统 |
| **自主性** | Autonomy | Agent 独立完成任务的程度（L1-L5） |
| **工具调用** | Tool Calling / Function Calling | LLM 通过 JSON Schema 调用外部 API |
| **ReAct** | Reasoning + Acting | 推理与行动交织的循环范式 |
| **思维链** | Chain-of-Thought (CoT) | 让 LLM 逐步推理的提示技术 |
| **思维树** | Tree-of-Thoughts (ToT) | 多路径搜索推理 |
| **反思** | Reflection | Agent 回顾自身行为并改进的机制 |
| **记忆流** | Memory Stream | 按时间线记录 Agent 所有经历 |
| **RAG** | Retrieval-Augmented Generation | 检索增强生成 |
| **MCP** | Model Context Protocol | Anthropic 提出的模型上下文协议 |
| **A2A** | Agent-to-Agent | Google 提出的 Agent 间通信协议 |
| **ACP** | Agent Communication Protocol | Agent 间标准化通信 |
| **多智能体** | Multi-Agent | 多个 Agent 协作完成任务的系统 |
| **规划** | Planning | Agent 分解任务、制定执行步骤 |
| **人机协同** | Human-in-the-Loop | 关键决策由人类确认 |
| **SWE-bench** | Software Engineering Benchmark | 编程 Agent 的标准评测 |
| **Guardrails** | Guardrails | Agent 的安全护栏/约束 |
| **CoALA** | Cognitive Architectures for Language Agents | 语言 Agent 认知架构理论 |
| **Orchestration** | Orchestration | 协调多个 Agent/工具的执行流程 |
| **Eval** | Evaluation | Agent 性能评测 |
